Neue Funktionen

Datei-Attribut

  • In diesem Attribut können pro Version eines Eintrages bis zu 100 Dateien gespeichert werden
  • Erstellen, Löschen, Umbenennen, Sortieren nach Name und Größe
  • Beim Herunterladen wird noch eine Warnung angezeigt, da Dateien noch nicht auf Viren gescannt werden
  • Viele Dateitypen können auch direkt angezeigt werden. Dazu zählen
    • Audio
    • Excel (Kann nicht auf möglichen Schadcode beim Anzeigen im Browser überprüft werden (XSS-Attacke))
    • Bilder
    • Video
    • PDF
    • Text
    • Markdown

Gemeinsames Caching zwischen Tabs

  • Daten zwischen mehreren offenen KoppaDb-Tabs eines Browser-Fensters werden nun geteilt, damit werden insbesondere unnötige Anfragen reduziert
  • Daten werden teilweise in Echtzeit und teilweise alle 10-12 Minuten automatisch aktualisiert. Alternativ können die Daten über den Synchronisieren-Knopf jederzeit aktualisiert werden. Ein komplettes Neuladen der Webseite ist theoretisch nur noch nötig, falls etwas nicht funktioniert.

Lokalisierungen

  • Zahlen werden nun entsprechend der ausgewählten Sprache formatiert.
    • Deutsch: 10.000,03
    • Englisch: 10,000.03
  • Daten werden nun entsprechend der ausgewählten Sprache formatiert.
    • Deutsch: So., 2. August 2020
    • Englisch: Sun, August 2, 2020
  • Die Datumsauswahl ist nun auch in Deutsch und Englisch verfügbar

  • Viele UX- und Design-Verbesserungen

  • Änderungen in einem Beziehungsattribut werden im Verlauf nun auch sofort ohne weitere Benutzerinteraktion geladen

Sicherheit

  • Man wird nun automatisch ausgeloggt, wenn man alle KoppaDb-Tabs schließt.
  • Mögliche XSS-Angriffsvektoren über den Markdown-Editor und -Viewer reduziert
  • Das Token wird nun nicht mehr im localStorage, sondern in der Laufzeit des SharedWorkers gespeichert. Somit wird es erschwert, XSS-Attacken ohne notwendige Benutzerinteraktion auszuführen.
  • Strengere CSP
